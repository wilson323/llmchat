【首次使用】

  1. 测试环境
     双击: test-env.bat
     或CMD: test-env.bat

  2. 调试启动
     双击: start-dev-debug.bat
     或CMD: start-dev-debug.bat

  3. 配置环境变量（如果需要）
     编辑: backend\.env

  4. 再次启动
     双击: quick-start.bat

────────────────────────────────────────────────────────────

【日常使用】

  快速启动（推荐）:
    双击: quick-start.bat
    或CMD: quick-start.bat

  标准启动:
    双击: start-dev.bat
    或CMD: start-dev.bat

────────────────────────────────────────────────────────────

【遇到问题】

  步骤 1: 环境测试
    test-env.bat

  步骤 2: 调试模式
    start-dev-debug.bat

  步骤 3: 查看文档
    TROUBLESHOOTING-WINDOWS.md
    WINDOWS-SCRIPTS.md

────────────────────────────────────────────────────────────

【常见问题快速修复】

  ✗ 脚本闪退
    → 使用 CMD 运行: start-dev-debug.bat

  ✗ concurrently 未安装
    → npm install

  ✗ 端口被占用
    → netstat -ano | findstr :3000
    → taskkill /PID <PID> /F

  ✗ 配置文件缺失
    → 运行 start-dev-debug.bat 自动创建

────────────────────────────────────────────────────────────

【访问地址】

  前端: http://localhost:3000
  后端: http://localhost:3001
  健康检查: http://localhost:3001/health

────────────────────────────────────────────────────────────

【停止服务】

  按 Ctrl+C 然后选择 Y
