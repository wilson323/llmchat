# LLMChat 术语表 v1.0.0

## 核心术语

| 英文 | 中文 | 代码示例 |
|------|------|---------|
| Agent | 智能体 | `AgentConfig` |
| Chat | 聊天 | `ChatWindow` |
| Conversation/Session | 会话 | `Conversation` |
| Message | 消息 | `ChatMessage` |
| Provider | 提供商 | `FastGPT` |
| Stream | 流式 | `streamResponse()` |

## 命名规范
- 变量/函数: camelCase
- 类/组件: PascalCase
- 常量: UPPER_SNAKE_CASE
- 文件: kebab-case

**规则**: 代码英文，文档中文，禁止混用。
