## 贡献指南

### 如何贡献
1. Fork 项目到你的 GitHub 账户
2. 创建功能分支 (`git checkout -b feature/amazing-feature`)
3. 提交你的更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建一个 Pull Request

### 开发规范
- 使用 TypeScript 开发新功能
- 遵循 ESLint 和 Prettier 规范
- 编写完整的测试用例
- 更新相关文档
- 确保所有测试通过

### 提交规范
- 使用语义化提交消息
- 每个提交应该是一个完整的逻辑单元
- 提交消息应该清晰地描述更改内容
- 关联相关的 Issue
